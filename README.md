# Weapon Detection System Using YOLO

## Overview
This project is a real-time weapon detection system built using YOLO, OpenCV, and PyTorch. The system can detect weapons such as pistols and knives from live camera feeds, images, and videos while distinguishing them from non-weapon objects.

The application provides visual alerts, bounding box annotations, FPS monitoring, and screenshot capture functionality, making it suitable for surveillance and security applications.

## Features
* Real-time camera detection
* Image detection
* Video detection
* Weapon alert system
* FPS monitoring
* Screenshot capture
* Automatic output generation
* YOLO-based object detection

## System Workflow
1. Load trained YOLO model.
2. Capture image, video, or camera stream.
3. Detect objects in each frame.
4. Classify object as weapon or non-weapon.
5. Draw color-coded bounding boxes.
6. Display real-time alerts.
7. Save results and screenshots.
